# Todo App (Beginner)

React + TypeScript で作られたシンプルなタスク管理アプリです。

## 機能

- タスクの追加
- タスクの完了/未完了の切り替え
- タスクの削除

## 技術スタック

- React 18
- TypeScript
- Create React App

## セットアップ

```bash
npm install
```

## 起動

```bash
npm start
```

http://localhost:3000 でアプリが開きます。

## スクリプト

| コマンド | 説明 |
|---------|------|
| `npm start` | 開発サーバーを起動 |
| `npm test` | テストを実行 |
| `npm run build` | 本番用にビルド |
